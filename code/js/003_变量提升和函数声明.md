### 函数提升
> 1.当函数名和变量名一样的时候，变量没有赋值，函数优先级高

```js
function a(){}
var a;
console.log(a)//function a(){}

```
> 2.当函数和变量名一样，变量已经赋值；声明的变量优先级高

```js
//①
function a(){}
var a=1;
console.log(a)//1

//②
var a =1;
function a(){}
console.log(a) // 1

```
> 3.函数声明的三种方式
```js
//1.函数表达式，在次之前调用会报错
var a = function(){}

//2.函数声明，可以在此代码前调用
function a(){}

//3.Function构造函数,(不推荐)
//Function构造行数最后一个参数为函数体，
var a = new Function("n","m","return n+m");

```
